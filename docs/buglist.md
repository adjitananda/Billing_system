## Инструкция по заполнению

1. При тестировании найдите ошибку
2. Определите её приоритет (🔴🟠🟡)
3. Заполните строку в соответствующей таблице
4. После исправления измените статус на ✅
5. Обновите дату в шапке файла

**Важно:** Разработчик перед началом работы должен сделать `git pull` и ознакомиться с актуальным баг-листом.
