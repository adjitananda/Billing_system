# Billing_DC - Биллинговая система для дата-центра

## Описание
Система для учёта и расчёта стоимости виртуальных машин в дата-центре. 
Позволяет отслеживать ресурсы ВМ, историю изменений конфигураций и 
ежедневно рассчитывать стоимость использования.

## Технологии
- Python 3.10+
- MySQL 8.0
- mysql-connector-python
- python-dotenv
